# 🤖 بوت تيليجرام - متجر منتجات رقمية

## 🚀 خطوات التشغيل

### 1. إنشاء البوت
- افتح تيليجرام وابحث عن `@BotFather`
- أرسل `/newbot`
- اختار اسم للبوت
- احفظ الـ **Token** اللي هيديك إياه

### 2. معرفة الـ Admin ID
- ابعت رسالة لـ `@userinfobot` على تيليجرام
- هيرد عليك بـ ID بتاعك (رقم زي 123456789)

### 3. تعديل الإعدادات
افتح ملف `config.py` وعدل:
```python
BOT_TOKEN = "ضع_التوكن_هنا"
ADMIN_IDS = [ضع_ID_هنا]
PAYMENT_VODAFONE = "رقم_فودافون_كاش"
PAYMENT_INSTAPAY = "@instapay_account"
```

### 4. تثبيت المتطلبات
```bash
pip install python-telegram-bot==20.7
```

### 5. تشغيل البوت
```bash
python bot.py
```
